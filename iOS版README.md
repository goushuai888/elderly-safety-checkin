# 死了么 - iOS应用

## 项目简介

"死了么"是一款专为独居老人设计的安全签到iOS应用。通过每日签到机制，确保老人安全，如果老人未按时签到，系统会发送本地通知提醒。

## 主要功能

### ✅ 已完成功能

1. **老人信息管理**
   - 添加/编辑/删除老人信息
   - 设置姓名、电话、住址
   - 自定义每日检查时间

2. **每日签到**
   - 简洁的一键签到界面
   - 大字体设计，适合老年人使用
   - 签到状态实时显示
   - 防止重复签到

3. **紧急联系人管理**
   - 为每位老人添加多个紧急联系人
   - 记录联系人姓名、电话、邮箱、关系
   - 方便的联系人列表管理

4. **签到记录统计**
   - 查看完整的签到历史
   - 统计总签到次数、最近7天、最近30天
   - 美观的数据可视化

5. **本地通知**
   - 每日定时提醒签到
   - 可自定义提醒时间
   - 使用iOS原生通知系统

6. **数据持久化**
   - 使用UserDefaults存储数据
   - 应用关闭后数据不丢失
   - 自动加载历史数据

## 技术栈

- **SwiftUI** - 现代化的UI框架
- **UserDefaults** - 本地数据存储
- **UserNotifications** - 本地通知提醒
- **Combine** - 响应式编程
- **iOS 15+** - 最低支持版本

## 使用说明

### 首次使用

1. 打开应用，点击"添加老人信息"
2. 填写老人的基本信息：
   - 姓名（必填）
   - 电话（必填）
   - 住址（选填）
   - 每日检查时间（默认20:00）

### 日常使用

1. **签到流程**：
   - 打开应用，选择对应的老人
   - 点击"点击签到"大按钮
   - 看到"今日已签到"提示即完成

2. **管理紧急联系人**：
   - 在签到界面点击"管理紧急联系人"
   - 点击右上角"+"添加联系人
   - 填写联系人信息并保存

3. **查看签到记录**：
   - 在签到界面点击"查看签到记录"
   - 可以看到详细的签到历史和统计数据

### 通知设置

1. 首次使用时会请求通知权限，请允许
2. 系统会在每天设定的时间发送签到提醒
3. 可以在"设置 > 通知"中管理应用通知
